A simple recent posts widget that displays recent posts with a featured image thumbnail.

== Description ==

JAM Recent Posts creates a simple recent posts widget that displays the post title, featured image thumbnail, and publish date. You can choose the number of posts displayed.

== Installation ==

Upload the JAM Recent Posts plugin to your blog, activate it, then use the widgets screen to configure the widget.
